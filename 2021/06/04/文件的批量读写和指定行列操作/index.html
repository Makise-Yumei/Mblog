文件指定行列数据的批量读写
2021-06-04 22:36:46 # Python

最近在尝试将风速当作工具变量时发现了几个很有意思的数据处理问题。

气象数据大多以众多txt集合的形式呈现（如下图），每一个txt都代表了某个气象站在某一段时期内的各项气象数值。想从中获得所有气象站中指定变量的均值/分位数最简单的方式就是一项项打开这些txt运行求解代码。但当气象站数量达到某一量级时，我们就难以使用人工方式进行遍历。

因此，我将在这篇博客里向你展示如何利用Python实现对多文件情况下指定行列数据的批量读写。

数据介绍

1.基本情况

这是2017年中国四百多个气象观测站点所提供的三百多万条地面气象数据。

  • 时间范围：大部分以三小时为单位，少部分有一小时数据。

  • 气象要素：气温、气压、露点、风向、风速、云量和降水量。

2.处理目标

提取各站的年平均风速。

数据处理

截取数据集中某一个气象站的地面气象数据为例。

第一列为年，第二列为月，第三列为日，第四列为小时，而第九列就是我们要提取的风速。每一列中或多或少出现的-9999则代表着缺失值。

我们可以将它看作一个以空白为分隔符的csv文件。

import os
import pandas as pd
import numpy as np

listx=[]
os.chdir('风速数据存储的绝对路径')
for filename in os.listdir(os.getcwd()): #遍历文件夹内所有文件
    listxappend(filename)

for i in range(412):
    f=pd.read_csv(listx[i],delim_whitespace=True,header=None) #以空白为分隔符读取文件
    f=f.replace(-9999,0) #处理缺失值
    a1=f.iloc[:,8] #指定第九列数据
    a2=np.mean(a1) #求第九列数据的平均值

print(a2)

运行代码，我们得到了这样的结果。

于是，我们成功地求出了2017年四百多个气象站的年平均风速。
